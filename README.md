# MAGI System v2.0

エヴァンゲリオンにインスパイアされた合議制AIシステム

## 概要

3つのAIユニットによる合議制分析システム:
- BALTHASAR-2: 創造的分析
- MELCHIOR-1: 論理的分析
- CASPER-3: 人間的分析

## 動作モード

- CONSENSUS: 合議制
- INTEGRATION: 統合分析
- SYNTHESIS: 創発的洞察

## クイックスタート


git clone https://github.com/miroqu369/magi-sys.git
cd magi-sys
npm install
node bootstrap.js

## テスト
./magi-cli.sh
