# DOGMA System v2.0

合議制AIシステム - 3つのAIユニットによる協調分析

## 概要

DOGMA Systemは3つの独立したAIユニットが協調して分析を行う合議制システムです。

### システムユニット
- **Unit-B2**: 創造的分析
- **Unit-M1**: 論理的分析
- **Unit-C3**: 人間的分析

## 動作モード
- **CONSENSUS**: 合議制による多数決
- **INTEGRATION**: 統合的分析
- **SYNTHESIS**: 創発的洞察

## クイックスタート
git clone https://github.com/miroqu369/dogma-sys.git
